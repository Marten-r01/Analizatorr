Analizator — это Full Stack приложение для загрузки, хранения и анализа генетических последовательностей в формате **FASTA**.

Пользователь загружает файл, система выполняет:
- валидацию входных данных
- парсинг FASTA
- нормализацию последовательности
- базовый биоинформатический анализ
- сохранение результатов
- визуализацию отчёта в web-интерфейсе

Проект предназначен для учебного и демонстрационного использования в области:
- биоинформатики
- full stack разработки
- алгоритмической обработки доменных данных

## 🔥 Актуальность

Проект актуален, поскольку задачи хранения, обработки и анализа генетических последовательностей имеют высокую практическую значимость в биоинформатике и образовании.

Разработка такого сервиса позволяет:
- автоматизировать первичный анализ FASTA-файлов
- сократить количество ручных операций
- уменьшить вероятность ошибок при обработке данных
- сделать результаты анализа наглядными и удобными для интерпретации

## 🎯 Востребованность

Проект востребован как учебный и прикладной инструмент для:
- студентов
- преподавателей
- начинающих исследователей
- разработчиков, изучающих биоинформатику

Сервис позволяет быстро выполнять базовый анализ последовательностей, сохранять результаты и визуализировать их в удобной форме.

## 🏗 Архитектура и стек

### Frontend
- Compose Multiplatform (Web/WASM)
- загрузка файлов
- отображение последовательности
- визуализация результатов анализа
- построение графиков

### Backend
- Kotlin 2.x
- Ktor
- PostgreSQL
- Exposed
- REST API
- JSON
- Multipart/form-data
- kotlinx.serialization
